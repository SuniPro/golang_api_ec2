# Golang 출석체크 API 


## 🖥️ 프로젝트 소개
Go Lang을 활용하여, 게임의 출석체크 기능을 제작하였습니다.
<br>

## 🕰️ 개발 기간
* 22. 10. 2. ~ 22. 10. 15.

### ⚙️ 개발 환경
- `Go lang`
- `GO orm`
- **IDE** : goland
- DB : mySQL

## 📌 주요 기능
#### 로그인
- JWT token 발급
#### 회원가입
- ID 중복 체크
- 자동 닉네임 생성
- 최종로그인 일자 확인
#### 출석체크
- 출석체크
- 일정 이상 일자 출석 시 보상
#### 리포트 등록
- 기타 도커 및 ec2 인프라 구축
